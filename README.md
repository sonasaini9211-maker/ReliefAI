ReliefAI

An Emergency Response and Resource Allocation System

---

Problem Statement

In emergency situations, response time is critical. However, delays in reporting, lack of prioritization, and inefficient resource allocation often lead to serious consequences.

---

Solution

ReliefAI is designed to improve emergency response by:

- Collecting incident reports from users
- Organizing and tracking incidents efficiently
- Assisting in resource allocation
- Providing a real-time monitoring dashboard for authorities

---

Features

- User Authentication (Login and Registration)
- Emergency Reporting System
- Real-time Dashboard
- Incident Tracking
- Location-based Data Handling
- Admin Panel

---

Technology Stack

- Frontend: HTML, CSS
- Backend: Flask (Python)
- Database: SQLite

---

How to Run the Project

Install dependencies:

pip install -r requirements.txt

Run the application:

python app.py

Open in browser:

http://127.0.0.1:5000
